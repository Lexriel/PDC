i=10 ; j = 91 ;

q = tab+j ; 
copy = (int *) malloc(q-p); 
if (copy) 
  for (tmp = copy; p < q; *(tmp++) = *(p++)) ; 
\end{verbatim}

tableau.

Expliquer pourquoi ce code n'est pas correct.
