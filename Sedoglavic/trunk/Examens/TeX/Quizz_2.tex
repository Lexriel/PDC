\`A quoi ressemblera le programme suivant apr\`es traitement par le pr\'eprocesseur~:
\begin{verbatim}
/* #include<stdio.h> */
   #define N 100 
   int main(void){
   #ifdef N
      printf("N vaut %d\n", N);
   #else
      printf ("N n'est pas defini\n");
   #endif
   return 0 ;
   }
\end{verbatim}
Pour r\'epondre, donnez le code~C r\'esultant de ce traitement.
